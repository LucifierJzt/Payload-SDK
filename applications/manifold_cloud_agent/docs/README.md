# Manifold Cloud Agent 文档

本目录存放生产应用专属文档，包括：

- 架构与通信协议决策；
- 配置与部署说明；
- 使用、运维和故障排查手册；
- 安全要求与控制指令安全规则；
- 实施计划和 TODO 跟踪。

## 计划文档

- `architecture.md`：组件边界、生命周期和数据流。
- `configuration.md`：云服务和 WebSocket 的配置结构。
- `deployment.md`：Manifold 上的构建与部署步骤。
- `operations.md`：监控、日志、恢复和故障处理。
- `TODO.md`：实施里程碑和待决事项。
